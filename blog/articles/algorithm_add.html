两个超大位数相加

算法
2025年11月8日 3分钟阅读 约52行

如何解决

这个算法核心原理是模拟人工竖式计算方法。首先将两个表示大数的字符串反转存储到整型数组中（个位在前），然后逐位相加并处理进位，最后将结果再次反转输出（高位在前）。

通过字符串存储和数组运算的方式，该算法突破了数据类型的位数限制，能够处理任意长度的整数加法，为高精度计算提供了基础。

代码实现

解决 C 
#include <stdio.h>
#include <string.h>

#define MAX_LEN 200

int main(void)
{
    char str1[MAX_LEN + 1], str2[MAX_LEN + 1];
    int num1[MAX_LEN] = {0}, num2[MAX_LEN] = {0}, result[MAX_LEN + 1] = {0};
    int len1, len2, max_len;
    
    scanf("%s", str1);
    scanf("%s", str2);
    
    len1 = strlen(str1);
    len2 = strlen(str2);
    max_len = (len1 > len2) ? len1 : len2;
    
    //将字符转换为数字,反转
    for (int i = 0; i < len1; i++)
    {
        num1[i] = str1[len1 - 1 - i] - '0';
    }
    for (int i = 0; i < len2; i++)
    {
        num2[i] = str2[len2 - 1 - i] - '0';
    }
    
    //逐位相加并处理进位
    int carry = 0;
    for (int i = 0; i < max_len; i++)
    {
        int sum = num1[i] + num2[i] + carry;
        result[i] = sum % 10;
        carry = sum / 10;//如果相加大于10，则carry+1，在下一位进位
    }
    
    //最后的进位
    if (carry > 0)
    {
        result[max_len] = carry;
        max_len++;
    }
    
    for (int i = max_len - 1; i >= 0; i--)
    {
        printf("%d", result[i]);
    }
    printf("\n");
    
    return 0;
}
